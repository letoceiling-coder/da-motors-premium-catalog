# Скрипты для парсинга автомобилей

## Описание

Скрипты для получения данных об автомобилях с сайта-донора `https://da-motors-msk.ru/catalog` через API и добавления их в каталог проекта.

## Скрипты

### `fetch-cars-from-api.js`

Основной скрипт для получения данных об автомобилях через API Tilda.

**Что делает:**
- Подключается к API endpoints сайта-донора
- Извлекает данные об автомобилях (марка, модель, цена, год, фото, характеристики)
- Автоматически исправляет некорректные цены (если цена > 100 млн, делит на 10000)
- Добавляет новые автомобили в файл `src/data/cars.ts`
- Удаляет дубликаты

**Извлекаемые данные:**
- Марка и модель автомобиля
- Цена (автоматически исправляется, если некорректная)
- Год выпуска
- Фотографии (из галереи товара)
- Характеристики: объем двигателя, мощность, тип топлива
- Статус: в наличии / в пути / под заказ

## Установка зависимостей

Все необходимые зависимости уже должны быть установлены. Если нет, выполните:

```bash
npm install
```

## Запуск

### Получение данных через API

```bash
npm run fetch-api
```

Или напрямую:

```bash
node scripts/fetch-cars-from-api.js
```

## Как это работает

1. Скрипт отправляет запросы к API endpoints Tilda
2. Получает HTML-ответы с встроенными JSON данными
3. Парсит JSON или HTML (fallback) для извлечения данных о товарах
4. Форматирует данные в структуру `Car`
5. Добавляет новые автомобили в `src/data/cars.ts` перед `export const brands`

## Настройка API endpoints

API endpoints настраиваются в начале файла `fetch-cars-from-api.js`:

```javascript
const API_ENDPOINTS = [
  {
    url: 'https://store.tildaapi.com/api/getproductslist/',
    params: {
      storepartuid: '756428026962',
      publickey: '...',
      // другие параметры
    }
  },
  // другие endpoints
];
```

## Автоматическое исправление цен

Скрипт автоматически исправляет некорректные цены:
- Если цена > 100 000 000, она делится на 10 000
- Это исправляет проблему, когда API возвращает цены в неправильном формате

## Удаление дубликатов

Скрипт автоматически удаляет дубликаты на основе комбинации:
- Марка
- Модель
- Цена

## Обработка ошибок

- Если API endpoint недоступен, скрипт продолжает работу с другими endpoints
- Если JSON не найден, используется fallback парсинг HTML
- Ошибки логируются в консоль, но не останавливают выполнение

## Логирование

Скрипт выводит в консоль:
- Прогресс запросов к API
- Количество найденных продуктов
- Количество уникальных автомобилей
- Результат добавления в каталог

## Важные замечания

1. **Резервное копирование**: Перед запуском рекомендуется сделать резервную копию файла `src/data/cars.ts`
2. **Проверка данных**: После запуска проверьте, что данные корректны
3. **Обновление endpoints**: Если структура API изменится, может потребоваться обновление endpoints
4. **Частота запуска**: Не запускайте скрипт слишком часто, чтобы не перегружать сервер донора

## Устранение проблем

### Скрипт не находит автомобили

- Проверьте, что API endpoints актуальны
- Убедитесь, что сайт-донор доступен
- Проверьте логи в консоли на наличие ошибок

### Цены некорректные

- Скрипт автоматически исправляет цены > 100 млн
- Если проблема сохраняется, проверьте формат данных в API ответе

### Дубликаты автомобилей

- Скрипт автоматически удаляет дубликаты
- Если дубликаты остаются, проверьте логику определения уникальности

### Ошибки парсинга JSON

- Скрипт использует fallback парсинг HTML
- Проверьте структуру HTML ответа, если парсинг не работает
