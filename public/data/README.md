# ⚠️ КРИТИЧЕСКИ ВАЖНО: НЕ УДАЛЯТЬ ЭТУ ДИРЕКТОРИЮ!

Эта директория содержит критически важные данные:

- `bot-config.json` - токен бота, настройки webhook
- `bot-users.json` - пользователи бота
- `bot-admins.json` - администраторы бота  
- `applications.json` - заявки из MiniApp
- `cars.json` - каталог автомобилей
- `webhook.log` - логи webhook

При деплое всегда:
1. Делайте резервную копию этой директории
2. Восстанавливайте её после клонирования репозитория
3. Проверяйте, что данные на месте
